üî¨ Portfolio as Systemic Exploration

My portfolio consists of interactive projects that investigate how rules, structures, and computational systems influence perception, decision-making, and engagement.

Machine Learning Connect 4

This project explores how different AI decision-making models affect player experience. By comparing Minimax and NEAT-based opponents, I studied how optimal versus imperfect systems change perceived fairness, tension, and emotional response. The project reflects my interest in using computational methods as creative variables rather than fixed solutions.

Procedural Generation Maze

This project examines procedural systems and spatial logic. Through algorithmic maze generation, I explored how randomness, repetition, and variation influence navigation and spatial understanding. Observing player behavior allowed me to reflect on how algorithmic structure translates into human experience.

2D Platformer Adventure

While visually simple, this project functions as a study of interaction systems. Movement physics, camera behavior, and combat rules were treated as interconnected variables that shape player expression. The focus was not on realism or spectacle, but on how computational parameters affect "game feel" and embodied interaction.

Together, these works demonstrate my interest in creative computation, where code, systems, and rules become expressive media.
