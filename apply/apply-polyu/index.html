My portfolio presents a series of design studies rather than finished products. Each project uses games and interactive systems as a medium to examine how users perceive, understand, and engage with designed systems.

2D Platformer Adventure

This project focuses on interaction design and user experience. Through iterative tuning of movement, camera behavior, and level structure, I explored how design decisions affect clarity, responsiveness, and player learning. The project reflects an emphasis on teaching through interaction rather than instruction, a principle that extends beyond games into general interaction design.

Procedural Generation Maze

This project examines spatial structure and readability. By observing player navigation behavior, I studied how layout variation, landmarks, and repetition influence orientation and confusion. These insights connect directly to foundational design concerns such as visual hierarchy, spatial organization, and guiding users through complex systems.

Machine Learning Connect 4

Although technically motivated, this project ultimately became a study of user perception and fairness.
